MySQL Server 安裝與登入成功教學（Windows）

✅ 一、下載與安裝 MySQL Server

  1. 前往官方下載頁：https://dev.mysql.com/downloads/installer/
  2. 下載「MySQL Installer for Windows」完整版本（含 Server、Workbench）
  3. 執行安裝程式，選擇「Developer Default」
  4. 依照安裝步驟設定 Root 密碼，並記住它

✅ 二、啟動 MySQL Server

  1. 安裝完成後，MySQL Server 通常會自動註冊為服務
  2. 你可以透過「服務（services.msc）」檢查是否正在執行
  3. 或使用命令提示字元確認：
    net start mysql

✅ 三、使用命令列登入 MySQL

mysql -u root -p

系統會提示輸入密碼，輸入你安裝時設定的 Root 密碼。

✅ 四、登入成功畫面

Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 10
Server version: 8.0.xx MySQL Community Server - GPL

mysql>

📌 常見補充說明

  • 若打不開 mysql 命令，請將 MySQL bin 路徑加入「環境變數」中的 PATH
  • 也可改用「MySQL Workbench」圖形化工具登入操作
  • 登入後可使用：SHOW DATABASES; 來檢查資料庫
