第 2 課：ORDER BY + LIMIT 排序與分頁

語法說明

SELECT 欄位 FROM 資料表
ORDER BY 欄位 [ASC | DESC]
LIMIT 筆數 OFFSET 起始位置;

ORDER BY 用來排序，LIMIT 搭配 OFFSET 可用來分頁。

查詢範例

查詢商品，價格從高到低排序：

SELECT * FROM products
ORDER BY price DESC;

查詢最便宜的前 1 筆商品：

SELECT * FROM products
ORDER BY price ASC
LIMIT 1;

查詢第 3~4 筆最貴商品：

SELECT * FROM products
ORDER BY price DESC
LIMIT 2 OFFSET 2;

模擬結果

id name price
2 泡麵 35.00
1 可樂 25.00

課堂練習

  1. 查詢價格最低的商品。
  2. 查詢價格從低到高排序的前 3 筆。
  3. 查詢價格大於 20 的商品，並照價格遞減排序。
  4. 查詢第 2~3 筆價格最低的商品。
  5. 練習 OFFSET 的作用，觀察跳過不同筆數的結果。

完成後，請將查詢語法貼上並驗證結果。
