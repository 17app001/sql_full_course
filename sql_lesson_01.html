第 1 課：SELECT + WHERE 查詢語法

語法說明

SELECT 欄位名稱 FROM 資料表名稱 WHERE 條件式;

可搭配各種比較運算符號，如 =、>、<、!= 等。

查詢範例

查詢所有商品：

SELECT * FROM products;

查詢價格大於 30 的商品：

SELECT * FROM products WHERE price > 30;

查詢名稱為「泡麵」的商品：

SELECT * FROM products WHERE name = '泡麵';

模擬結果

id name price
2 泡麵 35.00
3 麵包 20.00

課堂練習

  1. 查詢價格等於 25 的商品。
  2. 查詢價格小於或等於 30 的所有商品。
  3. 查詢商品名稱為「可樂」的商品資料。
  4. 試著用 != 查詢不等於「麵包」的商品。
  5. 想一想：<= 和 < 差在哪裡？

完成後，請將查詢語法貼上並測試結果是否正確。
